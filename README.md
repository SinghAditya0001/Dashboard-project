# Employee Management Dashboard

A React.js Employee Management Dashboard built using DummyJSON API.

## Features

- Fetch Employee Data
- Search Employees
- Filter by Department
- Filter by Gender
- Sort by Name and Age
- Add Employee
- Edit Employee
- Delete Employee
- Employee Details Modal
- Dashboard Statistics
- Employee Analytics
- Dark Mode
- Responsive Design

## Technologies Used

- React.js
- JavaScript
- CSS
- DummyJSON API
